# 1.传感器分类

| 类别          | 常见子类                                          | 关键指标（典型范围）                                         | 优点                         | 局限/注意                                       | 典型用途                                 |
| ------------- | ------------------------------------------------- | ------------------------------------------------------------ | ---------------------------- | ----------------------------------------------- | ---------------------------------------- |
| 摄像头        | 单目/双目/多目、鱼眼、深度（主动/被动）、事件相机 | 分辨率：1–8MP；FOV：60–200°；帧率：30–120FPS；带宽：百Mb/s–Gb/s | 信息量大、成本低、语义强     | 光照/眩光敏感，低纹理退化；外参与时间同步要求高 | 识别/跟踪、定位辅助、路面/货架检测、手眼 |
| 激光雷达      | 2D/3D                                             | 量程：10–200m；频率：5–20Hz；角分辨：0.1–0.5°                | 几何稳定、测距精确、易做SLAM | 雨雾/反射率影响；玻璃/黑物体；带宽高            | 建图/定位、障碍检测、测高/边缘           |
| 毫米波雷达    | 24/60/77GHz，成像/非成像                          | 量程：几十~两百米；速度精度优                                | 抗雨雾、测速准、穿透好       | 角分辨弱于相机/激光；多径                       | 室外避障、速度估计、冗余感知             |
| 超声/红外/ToF | 近距阵列、测距模组                                | 量程：0.1–5m；低带宽                                         | 低成本、补盲                 | 易受材质/角度影响                               | 贴近避障、边缘防护                       |
| 触觉/碰撞条   | 接触式                                            | 响应快                                                       | 只能“撞到才知道”             | 急停/防夹                                       |                                          |
| 卫星定位      | GNSS/RTK                                          | 室外厘米~米级                                                | 覆盖广                       | 室内无信号、遮挡                                | 室外导航/对齐高精地图                    |

# 2.相机分类

| 类别 | 核心原理                               | 主要优势                   | 主要短板                       | 典型应用                   |
| ---- | -------------------------------------- | -------------------------- | ------------------------------ | -------------------------- |
| 单目 | 一台相机成像                           | 便宜、语义强、生态成熟     | 无绝对尺度、光照敏感           | 检测/跟踪、VIO、低成本导航 |
| 双目 | 两相机                                 | 近中距深度、成本适中       | 远距误差放大、需严格同步与外参 | 室内避障/抓取、RGB-D替代   |
| 多目 | 三台及以上，环视/多基线                | 覆盖广、遮挡少、可全向SLAM | 外参网络复杂、带宽算力高       | 车载环视、园区巡检         |
| 鱼眼 | 超大FOV镜头（等距/等固角模型）150–220° | 视场超广、回环友好         | 畸变大、边缘分辨率低           | 盲区覆盖、窄通道通行       |
| 深度 | 红外光（结构光/ToF）来量距离           | 暗光好、近距准、ToF帧率高  | 强日光/多径/互扰               | 近距抓取/对接、避障        |

# 3.雷达分类

| 传感器     | 主要原理                    | 常见输出                      | 典型应用                   | 关键短板                        |
| ---------- | --------------------------- | ----------------------------- | -------------------------- | ------------------------------- |
| 激光雷达   | 主动光学                    | 2D 扫描/3D 点云、强度/回波    | 建图定位、避障、目标检测   | 雨雾/强反射影响、带宽大         |
| 毫米波雷达 | 毫米波                      | 距离-速度-角度目标点/稀疏点云 | 速度估计、全天候探测：雾天 | 角分辨率低于相机/激光，多径     |
| 超声       | 声波                        | 单束距离/阵列测距             | 近距避障、边缘防护、液位   | 材质/角度敏感、易受风噪         |
| 红外       | 被动热辐射 / 主动近红外反射 | 热成像/存在检测/近距深度      | 暗光检测、热异常、到位感知 | 强日光/反射干扰（主动IR）、热漂 |

下面这个例子里

- 激光雷达(LiDAR)：生成高精度的 3D 点云 数据。它能够**提供极高精度的深度信息和空间几何轮廓**，是构建车辆周围 3D 自由空间 和进行非标准障碍物检测的最可靠数据源。
- 摄像头：提供高分辨率的 2D 纹理和色彩数据。它是系统中唯一能够**提取丰富语义信息**的传感器，主要负责**车道线识别**、**交通标志/红绿灯读取**、以及**基于视觉大模型的目标分类（如区分行人与非机动车）**。
- 4D毫米波雷达
  - **全天候感知**：毫米波的波长使其能够穿透雨雪、沙尘和雾霾，不受光照和恶劣天气影响。
  - **直接测速**：基于多普勒效应，可直接输出目标的相对径向速度。
  - **高分辨率补盲**：由于具备俯仰角探测能力，4D 雷达可以区分静态路障（如高架桥墩、地面井盖），极大降低了传统毫米波雷达的误刹车率。
- 超声波雷达：负责极近距离（通常在 3 米以内）的盲区探测。主要用于低速场景下的自动泊车和**近场防碰撞预警**。

# 5.执行器

## 5.1 电机与传动

有刷电机+编码器+减速齿轮

| 类别      | 典型形态         | 优点                 | 局限               | 典型用法        |
| --------- | ---------------- | -------------------- | ------------------ | --------------- |
| BLDC/PMSM | 三相、FOC/六步   | 高效率、免维护、低噪 | 驱动复杂、成本中高 | 牵引/转向/云台  |
| 有刷DC    | H桥+PWM          | 便宜、易控           | 碳刷磨损、EMI      | 低成本执行/升降 |
| 步进      | 细分驱动         | 低速大力矩、开环可用 | 易失步/发热/效率低 | 轻载定位/丝杆   |
| 伺服系统  | 电机+编码器+驱动 | 高精度/高动态/力控   | 价格高、整定       | 关节/对位/插拔  |

## 5.2 末端执行与车用执行器

- **末端**：抓手（夹持/吸附/柔顺）、升降/顶升、夹爪/治具、云台/舵机。
- **车用执行**：转向执行器（角度闭环/力矩感知）、驱动（牵引轮对称/差速）、制动（电机/机械/电磁）。

# 6.控制器

## 6.1 计算与控制架构

- **主机（x86/ARM + GPU/NPU）**：感知/定位/规划/地图。
- 降级策略：L4->L3->L2 ,靠边停车
- **实时控制器（MCU/FPGA/PLC/RTOS）**：**1–5 ms** 控制回路（电流/速度/位置/力矩）、安全逻辑。
- **通信**：千兆以太网、**CAN/CAN-FD/EtherCAT**（控制）；QoS/时间敏感（TSN）按需。

## 6.2 控制算法与策略

- **低速/低阶**：PID + 前馈（速度/加速度）+ 抗饱和（反积分、输出限幅）。
- **多变量/有约束**：**MPC**（速度/加速度/曲率/障碍约束），或滑模/鲁棒控制。
- **差速底盘**：加速度、角速度。
- **安全状态机**：正常 → 退化（降感知/定位置信度低）→ 限速/爬行 → 安全停车；急停独立链路。
