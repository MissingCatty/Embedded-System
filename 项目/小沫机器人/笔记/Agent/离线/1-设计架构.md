# 1.为什么要板端微调

- 云端的通用大模型往往在特定领域或特定场景的专业度不够，需要对大模型进行微调。
- 局域网使用

- 数据不上传

# 2.端侧大模型和云端大模型的区别

由于端侧部署环境的资源有限，端侧只能选一些参数量较小的模型。因此，端侧模型不是特别智能，对于prompt的理解也没有云端大模型好，因此**输出不一定能够稳定且符合预期**。

---

**如何解决这个问题？**

即：如何让小参数量的模型能够稳定输出json格式的数据，所以要对模型进行处理。具体的手段有：**微调**。
