# 1.机器人的定义与分类

## 1.1 工程定义

机器人的工程定义：机器人是`感知-理解-决策-执行`的机电系统，具备一定的自主性和可编程性，能够对环境或对象施加有目的的作用。

与普通自动化设备相比，机器人强调：

- 不确定的环境中可以执行闭环行为
- 可重构/可编程
- 感知驱动（依赖传感器与状态估计）

自动驾驶分为几个层级：

- L2：低阶辅助驾驶-人开车 
- L3：高阶辅助驾驶-车自主开，但是人的手要放在方向盘上
- L4：一定范围内的自主驾驶-车自主开，不需要有人（远程操作）
- L5：任意范围内的自主驾驶

## 1.2 分类

一个维度无法描述清楚一个机器人。项目实践中建议用多维标签（形态、应用、作业空间、动力/执行、负载/尺寸 、人机协作模式）来刻画。

### 1.2.1 按形态/运动方式

- **轮式**：差速（两驱）、全向/麦克纳姆、汽车式（Ackermann）、舵轮
  - *优点*：结构简单、能效高、控制成熟；*局限*：越障/楼梯差
- **履带式**
  - *优点*：通过性与牵引力强；*局限*：效率低、对地面磨损
- **足式**：双足、四足、多足、**人形/仿人**（详见下节）
  - *优点*：环境适应性强、可跨越复杂地形；*局限*：控制与能耗挑战大
- **轮-腿/腿端小轮**（混合）
  - *优点*：兼顾能效与通过性；*局限*：机构与控制复杂
- **空中**：多旋翼、固定翼、垂直起降
  - *优点*：机动与覆盖范围大；*局限*：载荷/续航/法规约束
- **水面/水下、管道/蛇形、软体**
  - 面向特殊环境与任务的形态
- **机械臂/移动机械臂**
  - 固定/移动底座 + 末端执行器（夹爪、焊枪、喷头等）

**人形/仿人**

- **定义**：在形态与交互上仿人，通常具备头部传感、躯干、双臂、双足/轮式下肢。
- **子类**：纯双足人形、轮式人形、腿-轮混合、半身人形（固定/移动底座）。
- **特点**：适配人类环境与工具（门把手、楼梯、工位），便于人机共融；代价是**高自由度、高能耗与复杂控制**。

### 1.2.2 按应用

- **工业制造**：焊接、喷涂、搬运、装配、机床上下料
- **物流与仓储**：AGV/AMR、分拣、搬运、盘点
- **服务与商用**：清洁、配送、巡检、导览、安防
- **自动驾驶**：乘用车、无人物流车、矿卡/港口/园区重卡
- **医疗**：手术、康复、辅具、消毒
- **农业/户外作业**：喷洒、采摘、除草、播种
- **海事/能源/特种**：风电巡检、油气、消防、核辐照等

## 2.典型类别对照

| 形态类别                 | 代表场景                   | 优势                    | 主要短板             | 关键指标关注                           |
| ------------------------ | -------------------------- | ----------------------- | -------------------- | -------------------------------------- |
| 轮式（差速/全向/汽车式） | 仓储、配送、清洁、园区巡逻 | 简单高效、成本低        | 越障差、楼梯困难     | 速度、转弯半径、定位精度、避障可靠性   |
| 履带式                   | 工地、矿山、野外巡检       | 通过性强                | 效率低、磨损大       | 牵引力、地形适应、维护成本             |
| 四足/多足                | 林地、山地、救援           | 地形适应强              | 控制复杂、能耗高     | 步态稳定域、续航、负载比               |
| 人形/仿人                | 以人为本的室内/工位        | 环境/工具适配、人机共融 | 高 DOF、昂贵、续航短 | 全身力控稳定性、操作多样性、续航与散热 |
| 移动机械臂               | 柔性工厂、拣选/上下料      | “走+拿”一体、柔性强     | 规划复杂、稳定性挑战 | 末端定位/力控精度、基座定位            |
| 多旋翼                   | 巡检/测绘/应急             | 机动灵活                | 载荷与续航受限       | 续航、抗风、飞管合规                   |

# 3.AGV和AMR

| 维度          | AGV                             | AMR                                   |
| ------------- | ------------------------------- | ------------------------------------- |
| 导引方式      | 外部导引：磁条/地标/反光板/埋线 | 内部感知：LiDAR/相机 + SLAM/融合定位  |
| 路线/柔性     | 路线固定，改线成本高            | 路径自适应，改线=重建/更新地图        |
| 避障策略      | 遇人/障碍多为“停车等待”         | 动态避障，绕行与重规划                |
| 部署周期      | 场地施工/贴磁条/布反光板        | 扫描建图→参数调优，无需改造地面       |
| 适用场景      | 工艺固定、节拍稳定的刚性产线    | SKU/布局常变、叉车/人车混行的柔性场景 |
| 上线/维护成本 | 低到中（但改线贵）              | 中到高（但后期柔性强，TCO 可低）      |
| 依赖基础设施  | 高（导引设施与路权）            | 低（网络/充电/门禁等通用设施）        |
| 算法与算力    | 简单（巡线/站点逻辑）           | 复杂（SLAM、感知融合、规划控制）      |
| 典型安全      | 碰撞条/安全扫码器 + 固定路权    | 安全激光 + 感知冗余 + 语义避障/限速   |
