# 1. std_msgs/String.h

## 1.1 概念

`std_msgs` 是 ROS 里一个**标准消息包**。

你可以把它理解成：**ROS 官方提前帮你定义好的一批常用“数据类型”**，这样节点之间通信时，不用每次都自己写消息格式。

比如它里面有这些常见消息类型：

- `std_msgs/String`：字符串
- `std_msgs/Int32`：32位整数
- `std_msgs/Float64`：双精度浮点数
- `std_msgs/Bool`：布尔值
- `std_msgs/Header`：时间戳、坐标系这些元信息

## 1.2 数据格式

ROS 节点之间通过 `topic / service / action` 传数据。传的数据必须有固定格式，这个格式就是 `message`。

`std_msgs` 提供的就是最基础、最通用的那些消息格式。

例如你发一句话：

```cpp
std_msgs::String msg;
msg.data = "hello";
pub.publish(msg);
```

这里的 `std_msgs::String` 就表示：

- 这个消息类型来自 `std_msgs`
- 类型名叫 `String`
