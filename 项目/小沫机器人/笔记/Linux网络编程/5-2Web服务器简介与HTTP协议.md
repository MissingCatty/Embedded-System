# 1.Web服务器的功能

- 主要功能：接收 HTTP 请求，处理请求，返回 HTTP 响应。

- 不只是传文本，还能传：
  - HTML
  - 图片
  - 音频
  - 视频
  - JSON 数据
  - 文件

常见的Web服务器：Tomcat，Ngix

# 2.HTTP协议

## 2.1 报文格式

请求行 + 请求头 + 空行 + 请求体

### 2.1.1 请求行

| 请求方法 | URL（资源路径） | 协议版本 |
| -------- | --------------- | -------- |
| GET      | /index.html     | HTTP/1.1 |

### 2.1.2 请求头

是一行一行的键值对，例如：

```html
Host: www.example.com		// 要访问哪个主机
User-Agent: Mozilla/5.0		// 客户端是什么浏览器或程序
Accept: text/html			// 能接收什么类型的数据
```

### 2.1.3 空行

请求头后面必须跟一个空行。

它的作用是：

> 告诉服务器，请求头到这里结束了，下面如果还有内容，那就是请求体。

### 2.1.4 请求体

不是所有请求都有。

比如：

- GET 请求通常没有请求体
- POST 上传表单、文件时通常会有请求体

例如登录表单提交时，用户名密码就可能放在请求体里。
