下面两个函数配合使用：

创建子线程后，**主线程必须等待子线程执行完毕**，否则主线程退出会连带着子线程一起退出。

```c
#include <pthread.h>

int pthread_create(pthread_t *thread, const pthread_attr_t *attr, void *(*start_routine) (void *), void *arg);
/*
	- 功能：创建线程
	- thread：-O，新线程指针
	- attr：新线程属性，NULL表示使用默认属性
	- start_routine：线程执行函数
		- 接收一个void*参数
		- 返回一个void*
	- arg：传给start_routine函数的参数
	- 返回值：
		- 0：创建成功
		- 非0：创建失败，返回错误码
*/

int pthread_join(pthread_t thread, void **retval);
/*
	- 功能：等待指定线程结束
	- thread：线程指针
	- retval：接收子线程退出时返回的值
		- *retval取内容是一个void *，即一个地址值，thread_func通常返回一个（static
	值的）地址
	- 返回值：
		- 0：成功
		- 非0：失败，返回错误码
*/
```

---

**案例**：创建一个子线程，让它打印 1 到 5，主线程同时打印 100 到 104。

```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>

void *thread_func(void *arg)
{
    int n = *(int *)arg;
    for (int i = 0; i < n; i++)
    {
        printf("child thread print %d.\n", i);
        sleep(1);
    }
    return NULL;
}

int main(void)
{
    /* 创建子线程 */
    pthread_t thread_id;
    int n = 5;
    if (pthread_create(&thread_id, NULL, &thread_func, &n) != 0)
    {
        printf("child thread created error.\n");
    }

    /* 主线程打印100到104 */
    for (int i = 100; i < 105; i++)
    {
        printf("main thread print %d.\n", i);
        sleep(1);
    }

    /* 主线程等待子线程结束 */
    pthread_join(thread_id, NULL);
    printf("main thread %ld closed.\n", pthread_self());
}
```
