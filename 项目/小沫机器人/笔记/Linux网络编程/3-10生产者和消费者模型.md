# 1.概念

- **生产者**：负责产生数据
- **消费者**：负责处理数据

它们之间通过一个**缓冲区 / 队列**来交互。

# 2.模型的作用

生产和消费的速度，往往不一样。如果没有中间队列，就会出现：

- 生产太快，消费者来不及处理
- 消费者太快，但暂时没有数据可处理

所以这个模型的核心作用是：

**1. 解耦**

生产者只管放数据，消费者只管取数据。
 双方不需要知道对方内部怎么实现。

**2. 缓冲**

中间队列可以暂存数据，平滑速度差。

**3. 提高并发**

生产和消费可以同时进行，不必串行。

# 3.需要解决的几个问题

1. 对于生产者，缓冲区满了怎么办

   有效的处理方式如下：

   - 阻塞等待
   - 丢弃数据，返回失败

2. 对于消费者，缓冲区空了怎么办

   - 阻塞等待
   - 轮询重试

**以上两点可以用信号量来解决。**

1. 对于共享资源（缓冲区）的访问必须是互斥的

   > 你可以把生产者-消费者里的共享区想成这样几个变量：
   >
   > - 一个数组 `buf[]`
   > - 一个写位置 `tail`
   > - 一个读位置 `head`
   > - 一个元素个数 `count`
   >
   > 生产/消费都需要有两步：
   >
   > - 修改指针
   > - 修改元素内容
   >
   > 由于这两步不是原子性的，如果不上互斥锁，很容易一个线程修改元素内容，另一个元素修改指针，这就乱套了。

**以上这一点可以使用互斥量来解决。**

# 4.代码实现

案例：三个生产者，一个消费者

```c
#include <stdio.h>
#include <pthread.h>
#include <malloc.h>
#include <unistd.h>

typedef struct _Node
{
    int val;
    struct _Node *next;
} Node;

static Node *head = NULL, *tail = NULL;
pthread_mutex_t mutex;

Node *push(int val)
{
    if (head == NULL)
    {
        head = (Node *)malloc(sizeof(Node));
        head->val = val;
        head->next = NULL;
        tail = head;
    }
    else
    {
        tail->next = (Node *)malloc(sizeof(Node));
        tail = tail->next;
        tail->val = val;
        tail->next = NULL;
    }

    return tail;
}

int pop()
{
    if (head != NULL)
    {
        Node *tmp = head;
        head = head->next;
        int val = tmp->val;
        free(tmp);
        return val;
    }
    else
    {
        return -1;
    }
}

void traverse()
{
    Node *ptr = head;
    while (ptr)
    {
        printf("%d->", ptr->val);
        ptr = ptr->next;
    }
    printf("EOF\n");
}

void *consume(void *arg)
{
    while (1)
    {
        pthread_mutex_lock(&mutex);
        if (pop() < 0)
        {
            printf("queue empty, pop failed.\n");
        }
        else
        {
            printf("consumer pop, current queue: ");
            traverse();
        }
        pthread_mutex_unlock(&mutex);
        sleep(1);
    }
    return NULL;
}

void *produce(void *arg)
{
    static int num = 0;
    while (1)
    {
        pthread_mutex_lock(&mutex);
        push(num++);
        printf("producer push, current queue: ");
        traverse();
        pthread_mutex_unlock(&mutex);
        sleep(5);
    }
    return NULL;
}

int main(void)
{
    pthread_mutex_init(&mutex, NULL);

    pthread_t consumer1, prod1, prod2, prod3;
    pthread_create(&consumer1, NULL, consume, NULL);
    pthread_create(&prod1, NULL, produce, NULL);
    pthread_create(&prod2, NULL, produce, NULL);
    pthread_create(&prod3, NULL, produce, NULL);

    pthread_detach(prod3);
    pthread_detach(prod2);
    pthread_detach(prod1);
    pthread_detach(consumer1);

    while (1)
    {
        sleep(10);
    }

    free(head);
    pthread_mutex_destroy(&mutex);
}
```
