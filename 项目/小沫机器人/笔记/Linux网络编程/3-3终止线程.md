```c
#include <pthread.h>

void pthread_exit(void *retval);
/*
	- 功能：结束当前线程，线程退出值设为 retval
	- retval：当前线程退出时带出的返回值，供 pthread_join 接收
*/
```

和return的效果差不多，

```c
void *thread_func(void *arg)
{
    static int x = 10;
    return &x;
}

void *thread_func(void *arg)
{
    static int x = 10;
    pthread_exit(&x);
}
```

`int pthread_join(pthread_t thread, void **retval)`中的retval实际上是直接把&x（x的地址）返回出来，没有先解包（因为不知道是什么类型的指针）
