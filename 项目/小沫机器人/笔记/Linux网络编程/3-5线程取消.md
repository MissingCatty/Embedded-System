线程取消主要使用的是`pthread_cancel`函数：

```c
#include <pthread.h>

int pthread_cancel(pthread_t thread);
/*
	- 功能：向指定线程发送“取消请求”，让线程终止
	- 返回值：
		- 0：成功发送取消请求
		- 非0：失败，返回错误码
*/
```

目标线程是否马上结束，要看它的**取消状态**和**取消类型**。

默认情况下，大多数线程是：

- **允许取消**：`PTHREAD_CANCEL_ENABLE`
- **延迟取消**：`PTHREAD_CANCEL_DEFERRED`

这表示：收到 `pthread_cancel` 请求后，不一定立刻退出，通常要等线程运行到某个**取消点**时才会真正结束。常见取消点有：

- `sleep()`
- `read()`
- `write()`
- `pthread_join()`
- `pthread_cond_wait()`

上述取消点都是在被取消线程中的，而不是执行取消函数的那个线程。
