# 1.概念

如果有多个线程想要对同一个资源进行**读操作**，按照道理是不需要互斥访问的。

这种情况下，如果使用互斥锁，就会导致一个线程读取完毕之后其他线程才能读，没有必要。

- 总结一句话：写时互斥，读时不互斥

读写锁：

- 如果有一个线程对资源进行**读操作（上读锁）**，则允许其他线程**读（允许上读锁）**，但不允许**写（上写锁阻塞）**
- 如果有一个线程对资源进行**写操作（上写锁）**，则不允许其他线程**读写（允许上读/写锁）**
- 一个线程对资源读，同时有另一个线程要对资源写，则**写的优先级高**

# 2.函数

## 2.1 创建读写锁

```c
int pthread_rwlock_init(
    pthread_rwlock_t *restrict rwlock,
    const pthread_rwlockattr_t *restrict attr
);
/*
    - 功能：初始化一个读写锁
    - rwlock：要初始化的读写锁
    - attr：属性（一般传 NULL，表示默认）
    - 返回值：
        - 成功：0
        - 失败：错误码（不是 -1）
*/
```

## 2.2 销毁一个读写锁

```c
int pthread_rwlock_destroy(pthread_rwlock_t *rwlock);
/*
    - 功能：销毁一个读写锁
    - rwlock：要销毁的读写锁
    - 注意：销毁前，读写锁必须已经不再被任何线程持有
    - 返回值：
        - 成功：0
        - 失败：错误码
*/
```

## 2.3 加读锁

### 2.3.1 阻塞方式加锁

```c
int pthread_rwlock_rdlock(pthread_rwlock_t *rwlock);
/*
    - 功能：以阻塞方式加读锁
    - rwlock：要加锁的读写锁
    - 特点：多个线程可以同时持有读锁
    - 返回值：
        - 成功：0
        - 失败：错误码
*/
```

- 说明：
  - 如果没有线程持有写锁，则当前线程加读锁成功
  - 如果有线程持有写锁，则当前线程阻塞等待

### 2.3.2 非阻塞方式加锁

```c
int pthread_rwlock_tryrdlock(pthread_rwlock_t *rwlock);
/*
    - 功能：以非阻塞方式尝试加读锁
    - rwlock：要加锁的读写锁
    - 返回值：
        - 成功：0
        - 失败：错误码
*/
```

- 说明：
        - 如果当前可以加读锁，则立即成功返回
        - 如果不能加读锁，则立即失败，不阻塞

## 2.4 加写锁

### 2.4.1 阻塞方式

```c
int pthread_rwlock_wrlock(pthread_rwlock_t *rwlock);
/*
    - 功能：以阻塞方式加写锁
    - rwlock：要加锁的读写锁
    - 返回值：
        - 成功：0
        - 失败：错误码    
*/
```

- 说明：只有当没有线程持有读锁或写锁时，当前线程才能加写锁，否则阻塞等待

### 2.4.2 非阻塞方式

```c
int pthread_rwlock_trywrlock(pthread_rwlock_t *rwlock);
/*
    - 功能：以非阻塞方式尝试加写锁
    - rwlock：要加锁的读写锁
    - 返回值：
        - 成功：0
        - 失败：错误码
*/
```

- 说明：
        - 如果当前可以加写锁，则立即成功返回
        - 如果不能加写锁，则立即失败，不阻塞

## 2.5 解锁读写锁

```c
int pthread_rwlock_unlock(pthread_rwlock_t *rwlock);
/*
    - 功能：解锁读写锁
    - rwlock：要解锁的读写锁
    - 返回值：
        - 成功：0
        - 失败：错误码
*/
```

- 说明：既可以解读锁，也可以解写锁

# 3.案例

- 一个全局变量 `number` 表示共享数据
- 3 个线程负责“读”
- 1 个线程负责“写”
- 读线程之间可以同时读
- 写线程写的时候，别人不能读也不能写

```c
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <pthread.h>

static int number = 10;
pthread_rwlock_t rwlock;

void *read_func(void *arg)
{
    while (1)
    {
        // 上读锁
        pthread_rwlock_rdlock(&rwlock);
        printf("thread %ld is reading, number = %d.\n", pthread_self(), number);
        pthread_rwlock_unlock(&rwlock);
        sleep(1);
    }
    
    return NULL;
}

void *write_func(void *arg)
{
    while (1)
    {
        // 上写锁
        pthread_rwlock_wrlock(&rwlock);
        printf("thread %ld is writing ...\n", pthread_self());
        number++;
        pthread_rwlock_unlock(&rwlock);
        sleep(1);
    }

    return NULL;
}

int main(void)
{
    pthread_t tid_r1, tid_r2, tid_r3, tid_w1;

    // 初始化读写锁
    if (pthread_rwlock_init(&rwlock, NULL) != 0)
    {
        printf("rwlock init error.\n");
        exit(1);
    }

    // 创建三个读线程，一个写线程
    pthread_create(&tid_r1, NULL, read_func, NULL);
    pthread_create(&tid_r2, NULL, read_func, NULL);
    pthread_create(&tid_r3, NULL, read_func, NULL);
    pthread_create(&tid_w1, NULL, write_func, NULL);

    // 释放线程
    pthread_join(tid_r1, NULL);
    pthread_join(tid_r2, NULL);
    pthread_join(tid_r3, NULL);
    pthread_join(tid_w1, NULL);

    // 释放读写锁
    pthread_rwlock_destroy(&rwlock);
}
```
