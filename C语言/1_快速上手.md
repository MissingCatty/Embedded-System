# 1. C语言的参数

- 数组作为参数：以**引用**的形式进行传递
- 标量和常量作为参数：以**值**的形式进行传递

# 2. 正确的注释方式

```c
/*
** 注释第一行
** 注释第二行
*/
void function(int param)
{
    //...
}
```

# 3. 常用printf格式代码

| 格式 |    含义    |
| :--: | :--------: |
|  %d  | 10进制整数 |
|  %o  | 8进制整数  |
|  %x  | 16进制整数 |
|  %g  |   浮点值   |
|  %c  |    字符    |
|  %s  |   字符串   |

# 4. 常用scanf格式代码

| 格式 |         含义         | 变量类型 |
| :--: | :------------------: | :------: |
|  %d  |     读取一个整数     |   int    |
| %ld  |    读取一个长整型    |   long   |
|  %f  |    读取一个浮点数    |  float   |
| %lf  | 读取一个双精度浮点数 |  double  |
|  %c  |     读取一个字符     |   char   |
|  %s  |    读取一个字符串    | char数组 |
