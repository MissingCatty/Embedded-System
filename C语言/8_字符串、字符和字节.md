# 1 字符串长度

```c
#include <string.h>

size_t strlen(char const *str);
```

注意：`size_t`是一个**无符号整型**，使用时应当注意

```c
strlen(x) > strlen(y);
strlen(x) - strlen(y) > 0;
```

上述的两个表达式的结果是不一定相等的，因为`strlen`的返回值是一个无符号整型，两个无符号整型加减的结果也是无符号整型，所以当`x`的长度小于`y`时，二者相减是大于0的，与预期不符

**警告**：如果**无符号整型和有符号整型放在一起运算**，通常在逻辑上肯定是想结果是有符号的，然而在一些编译器中会将有符号类型转为无符号类型进行计算，所以为了保险起见，在运算时**首先将无符号类型转为有符号类型**

# 2 常用字符串函数

## 2.1 复制字符串

```c
char *strcpy(char *dst, char const *src);
```

`src`是一个指针常量，意味着`src`指向一块固定的内存，不能修改

执行过程为`src`的元素（包括`'\0'`）依次占用`dst`中的元素

```c
char s1[] = "Original message";
strcpy(s1, "Different");
```

执行之后，`s1`将变为

![image-20241010210932308](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20241010210932308.png)

注意：要保证`dst`的长度比`src`要长，否则，`src`将侵占`dst`之后的空间，从而改变原先的内容，造成未知的错误

返回值为`dst`

```c
char *strncpy(char *dst, char const *src, size_t len);
```

复制`len`个字符到`dst`里，**严格按照`len`的长度执行复制**，不考虑`dst`是否能够容纳。同时，复制过程将包含`dst`中原本的`null`终止符。

也就是说，如果原本`dst`包含结尾`null`的总长度是10，此时如果指定`len`恰好也等于10，那么就会出现将结尾`null`覆盖的问题，造成**内存泄漏**。

如果指定的`len`比`src`包含`null`的长度要大，则将超出的部分用`null`代替；否则，直接复制`len`个字符（包括`null`）。例如：

```c
char dst[] = "----------";      // dst包含null的长度为：11
char src[] = "12345";           // src包含null的长度为：6

char *res = strncpy(dst, src, 5);
printf("%s\n", res);

res = strncpy(dst, src, 6);
printf("%s\n", res);

/*	输出
**	12345-----
**	12345
*/
```

## 2.2 连接字符串

```c
char *strcat(char *dst, char const *src);
```

将`src`的内容连接到`dst`后面，所以需要保证`dst`的空间足够容纳连接后的字符串

返回值是`dst`

```c
char *strcat(char *dst, char const *src, int len);
```

执行过程：从`dst`的末尾`null`开始替换，如果拼接后有效长度为`dst`总长度-1，则在最后添上`null`；否则，就会产生溢出，同时不添加`null`

## 2.3 字符串比较

```c
int strcmp(char const *str1, char const *str2);
```

两个字符串每个字符依次比较，比较到第一个不相同的字符（包括`'\0'`，即某一个字符串比较短）时，

如果`str1`的ASCII码比`str2`小，则返回`-1`；

如果`str1`的ASCII码大，则返回`1`

两字符串完全相同返回`0`

```c
int strncmp(char const *str1, char const *str2, int len);
```

比较前`len`个字符

## 2.4 查找一个字符

```c
char *strchr(char const *str, int ch);
char *strrchr(char const *str, int ch);
```

第1个函数查找字符`ch`第一次出现的位置

第2个函数查找字符`ch`最后一次出现的位置

如果没找到，则返回`null`

## 2.5 查找一个集合中的字符

```c
char *strchr(char const *str, char const *group);
```

`group`是一个字符集合，将待查字符全放在里面，返回第一次碰到的`group`中的字符位置

## 2.6 查找子串

```c
char *strstr(char const *s1, char const *s2);
```

## 2.7 前缀匹配

```c
size_t strspn(char const *str, char const *group);
```

从`str`第一个开始，判断字符是否在`group`里，返回前缀由`group`中字符组成的最大长度

如：

```c
char str1[] = "-1,3,4,12,4,1,hello";
int len1 = strspn(str1, "0123456789");

char str2[] = "145,4,12,4,1,hello";
int len2 = strspn(str2, "0123456789");

/*
**	len1 = 0
**	len2 = 3
*/
```

```c
size_t strcspn(char const *str, char const *group);
```

功能与`strspn`相反，找不在`group`里的最长前缀长度

## 2.8 分隔字符串

```c
char *strtok(char *str, char const *sep);
```

使用分隔符`seq`进行分割，使用方法是：

1. 将原字符指针`str`赋值给一个静态指针`last`，表示程序执行期间该指针不会消失

2. 移动`last`指针，第一次遇到一个`sep`分隔符就停止，将分隔符替换为`\0`，同时返回分隔符前半部分内容
3. 如果想要继续分割，就传入一个`NULL`给`str`参数，表示继续之前的分割，而不是分割一个新的字符串
4. 如果没有遇到分隔符，`last`指针就会一直移到`NULL`为止，并返回`NULL`之前的字符串

```c
char str[] = "145,4,12,4,1,hello";
char sep[] = ",";
char *p = str;
while(char *token = strtok(p, sep))
{
    if(token)
    {
        printf("%s\n", token);
        p = NULL;
    }
}
```

# 3 内存操作

对内存的操作和对字符串的操作类似，不过对字符串的处理时，遇到`NULL`就会停止，因为字符串内部不允许出现`NULL`，而一般的数组中出现`0`是很正常的情况

**内存复制**

```c
void *memcpy(void *dst, void const *src, size_t len);
```

（`void *`是未定指针类型，任何指针都可以转换为`void *`）

**内存比较**

```c
int memcmp(void const *a, void const *b, size_t len);
```

**查找第一次出现的值**

```c
void *memchr(void const *a, int ch, size_t length);
```

**内存设置值**

```c
void *memset(void *a, int ch, size_t length);
```
