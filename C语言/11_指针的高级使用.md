# 1 函数指针

## 1.1 函数指针的声明

```c
int *func();
int (*func)();
```

- 第一行声明了一个返回值为`int`指针的函数
- 第二行声明了一个函数指针，它所指向的单位返回一个`int`类型

## 1.2 函数指针的用法

函数指针和其他指针一样，在声明后不能立刻使用，必须进行初始化，将其指向另一个函数

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    int (*operation)(int, int);         // 声明一个形参为int,int的函数指针
    operation = &add;                   // 指针绑定
    int result = (*operation)(1,1);     // 使用函数指针
    printf("%d", result);
}
```

### 1.2.1 回调函数

- 定义：将函数本身作为参数，传递给另一个函数，后者“回调”传入的函数，所以叫回调函数

```c
#include <stdio.h>

// 定义一个函数，它接受一个回调函数作为参数
void process(int (*callback)(int, int), int a, int b) {
    int result = callback(a, b);  // 调用回调函数
    printf("Result: %d\n", result);
}

// 定义一个具体的回调函数
int add(int x, int y) {
    return x + y;
}

int main() {
    // 将 add 作为回调函数传递
    process(add, 3, 4);
    return 0;
}
```

### 1.2.2 转移表

```c
int (*func[N])();
```

该语句声明了一个名为`func`的包含`N`个函数指针的数组，具体使用方法如下：

```c
int Add(int a, int b) {
	return a + b;
}

int Sub(int a, int b) {
	return a - b;
}

int Mul(int a, int b) {
	return a * b;
}

int Div(int a, int b) {
	return a / b;
}

int(*p[5])(int, int) = { 0,Add,Sub, Mul,Div };

int ret = p[input](x, y); 
```
