# 1 整型函数

## 1.1 算数函数<stdlib.h>

```c
int			abs(int value);
long int	labs(long int value);
div_t		div(int numerator, int denominater);
ldiv_t		ldiv(long int numer, long int denom);
```

`div_t`是一个结构体，包含了`int quot`和`int rem`两个成员，分别表示商和余数

## 1.2 随机数<stdlib.h>

```c
int		rand(void);
void	srand(unsigned int seed);
```

- `rand`函数返回一个在0到RAND_MAX之间的伪随机数，想要获得某个范围内的随机数，就对这个范围大小取模

  - 需要注意的是：每次运行生成的随机数序列是相同的

    ```c
    for (int i = 0; i < 10; i++) {
        printf("%d ", rand() % 100);
    }
    
    // 每次生成的10个随机数总是相同的
    ```

- `srand`函数是带`seed`的伪随机数，可以**避免每次运行时获得相同的随机数序列**。

  - 通常使用每天的时间作为`seed`

    ```c
    srand((unsigned int)time(0))
    ```

## 1.3 字符串转换<stdlib.h>

字符串转换函数**把字符串转为数值**。

```c
int			atoi(const char *string);	//ASCII to Integer
long int	atol(const char *string);
```

- 转换过程中，会忽略字符串开头的空白字符（如空格、tab等）
- 遇到第一个非数字字符时，会终止转换
- 不处理错误情况，如果字符串不是有效的数字，返回值不可靠（如输入`123abc`，返回`123`，而没法检查是否遇到了不可转换的字符）

```c
long int			strtol(char const *string, char **endptr, int base);
unsigned long int	strtoul(char const *string, char **endptr, int base);
```

- `endptr`: 指向第一个不能转换的字符的指针。如果不需要此信息，可以传 `NULL`。
- `base`: 进制（通常为 10）。**如果传 0，则自动检测进制**：以 `0x` 开头的字符串被解释为十六进制，以 `0` 开头的字符串解释为八进制，其他情况为十进制。

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    const char *str = "12345";
    char *endptr;
    long num = strtol(str, &endptr, 8);  // 8表示十进制
    printf("数字是: %ld\n", num);

    const char *strWithChars = "123abc";  // 遇到非数字字符会停止转换
    num = strtol(strWithChars, &endptr, 10);
    printf("数字是: %ld\n", num);
    printf("停止转换的位置是: '%s'\n", endptr);  // endptr 指向第一个非数字字符

    const char *hexStr = "0x1A3F";  // 十六进制字符串
    num = strtol(hexStr, &endptr, 0);  // 进制为0时，会根据前缀判断
    printf("十六进制转换的结果是: %ld\n", num);

    return 0;
}

/*
 * 数字是: 5349
 * 数字是: 123
 * 停止转换的位置是: 'abc'
 * 十六进制转换的结果是: 6719
 */
```

## 2.4 浮点表示形式<math.h>

用规定的格式表示一个浮点数

```c
double frexp(double value, int *exponent);
double ldexp(double fraction, int *exponent);
double modf(double value, double *ipart);
```

- `frexp`：$fraction*2^{exponent}=value$，$0.5\le fraction\lt1$，返回`fraction`的值
- `ldexp`：返回`value`的值
- `modf`：把一个浮点数分成**整数部分**和**小数部分**，整数部分存放在`ipart`里，小数部分作为返回值返回

# 3 时间和日期函数

## 3.1 处理器时间<time.h>

```c
clock_t clock(void);
```

返回：从程序开始执行处理器所消耗的时间（处理器时钟滴答的次数，通常需要除以`CLOCKS_PER_SEC`转为秒）

近似估计，非精确值。如果想要获取精确值，可以在程序开始时调用一次，结束时调用一次，两次相减

## 3.2 当天时间<time.h>

```c
time_t time(time_t *returned_value);
```

返回当前的时间和日期，如果参数非`NULL`，那么时间就保存在这个指针里。如果机器无法提供当前时间和日期，或时间值太大（超过了`time_t`的范围），就返回`-1`

**返回的结果是时间的编码方式，并不一定是秒**，不同的编译器中可能有不同的涵义。一种常见的表示时间的方式是从一个任选的时间点开始流逝的秒数。在MS-DOS和UNIX系统中，这个时间是1970.1.1 00:00:00

## 3.3 时间类型操纵函数

### 3.3.1 ctime

```c
char *ctime(time_t const *time_value);
```

将一个`time_t`的指针所指向的时间值，解析成一个格式为`Sun Jul 4 04:02:27 1976\n\0`字符串。这个字符串是一个固定位置的静态数组，所以**每一次调用该函数，会将上一次调用的值覆盖掉**。

### 3.3.2 difftime

```c
double difftime(time_t time1, time_t time2);
```

计算`time`和`time2`的差，并把结果转为秒。

### 3.3.7 strftime

```c
size_t strftime(char *string, size_t maxsize, char const *format, struct tm const *tm_ptr);
```

将`tm`结构体按照自定义的格式，生成一个字符串。

如果结果的字符串长度小于`maxsize`，则将转换结果保存在`string`数组里，返回字符串的长度；否则，返回`-1`

![image-20241027224356623](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20241027224356623.png)

```c
#include <stdio.h>
#include <time.h>

int main() {
    time_t t = time(NULL);  // 获取当前时间
    struct tm *tm_info = localtime(&t);  // 将时间转换为当地时间的 tm 结构

    char buffer[80];  // 用于存储格式化后的时间字符串

    // 格式化时间
    strftime(buffer, sizeof(buffer), "%Y年%m月%d日 %H:%M:%S", tm_info);

    // 输出结果
    printf("当前时间: %s\n", buffer);

    return 0;
}

/*
** 2024年10月27日 14:45:30
*/
```
