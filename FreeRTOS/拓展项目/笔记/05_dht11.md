# 1.通信原理

dht11采用单总线的通信方式与主机通信，只有在接收到开始信号后才会触发温湿度采集。

**传输格式**

一次传输5个字节，传输时，高位在前，低位在后。

数据格式为：

> 湿度整数数据(8bit) + 湿度小数数据(8bit) + 温度整数数据(8bit) + 温度小数数据(8bit) + 校验和(8bit)

- 湿度小数部分（DHT11 恒为 0）
- 8bit 温度小数部分（DHT11 恒为 0）
- 校验和（前 4 个字节的和）

注：在传输数据时，需要严格按照上述时序，不允许打断。

**硬件连线**

dht11只需要一根DATA线与主机相连，且需要**外接上拉电阻**。

**传输时序**

1. **主机请求**

   主机需要通过单总线向 DHT11 发送一个起始信号，以触发数据传输：

   - **主机拉低总线**（起使信号）：主机将 **数据线拉低**（低电平 0） **至少 18ms**（确保 DHT11 能够检测到请求）
   - **主机释放总线**（释放控制权）：主机将数据线 **拉高** **20~40µs**，进入等待状态，准备接收 DHT11 的响应信号

   **总结来说，就是主机先拉低总线18ms（先控制总线），之后拉低30µs（释放总线，总线回归空闲态）。**

2. **DHT11响应**

   DHT11在检测到主机的起始信号后，会进行如下操作：

   - **DHT11 拉低数据线**（响应信号）：DHT11 **主动拉低** 数据线 **80µs**，表示已经收到主机的请求
   - **DHT11 拉高数据线**（准备发送数据）：DHT11 **主动拉高** 数据线 **80µs**，准备开始传输温湿度数据

3. **数据传输**

   DHT11 发送 **40bit（5字节）** 的数据给主机，**单个数据位**传输方式：

   - DHT11 拉低总线 50µs（**起始信号**）
   - **发送 0**：数据线拉高 **26~28µs** 后拉低
   - **发送 1**：数据线拉高 **70µs** 后拉低

# 2.代码实现

**起始信号**

```c
void _dht11_start(void)
{
    MASTER_WRITE(0);
    delay_ms(18);		// 控制总线18ms
    MASTER_WRITE(1);
    delay_us(30);		// 释放总线30us
}
```

这里说是30us，但其实对主机来说，只要释放总线后等待从机响应即可。

**接收响应**

从机接收到主机请求，首先会拉低总线80us，所以

1. 首先主机会等待总线被从机拉低，并设定一个超时时间（如100us），如果低电平持续时间没有超过100us，则转2；
2. 之后等待总线被从机拉释放（高电平），同样设定一个超时时间，如果高电平持续时间没有超过100us，即在100us内总线被从机拉低，说明数据开始传输；

```c
uint8_t _dht11_check_response(void)
{
    // 等待低电平的到来
    uint8_t timeout = DHT11_TIMEOUT;
    while (MASTER_READ())
    {
        delay_us(1);
        timeout--;
        if (!timeout)
            return 0;
    }

    // dht11拉低响应（持续80us左右）
    timeout = DHT11_TIMEOUT;
    while (!MASTER_READ())
    {
        delay_us(1);
        timeout--;
        if (!timeout)
            return 0;
    }

    // dht11拉高释放总线（持续80us左右）
    timeout = DHT11_TIMEOUT;
    while (MASTER_READ())
    {
        delay_us(1);
        timeout--;
        if (!timeout)
            return 0;
    }
    return 1;
}
```

**接收数据**

在接收每一位数据之前，主机需要等待从机拉低总线一段时间（50us），之后不管传输是0还是1，总线一开始总要被拉高一段时间，只不过拉高的时间不同。

在规定超时时间内等到低电平，就说明是下一次传输开始。

```c
uint8_t dht11_receive_data(void)
{
    // 开启通信
    _dht11_start();

    // 接收响应
    if (!_dht11_check_response())
    {
        return 1; // 响应超时
    }

    // 接收数据
    uint8_t data[5] = {0x00};
    uint8_t timeout;

    for (uint8_t i = 0; i < 5; i++)
    {
        for (uint8_t j = 0; j < 8; j++)
        {
            // 读取每一位之前，等待dht11拉低约50us
            timeout = DHT11_TIMEOUT;
            while (!MASTER_READ())
            {
                delay_us(1);
                timeout--;
                if (!timeout)
                    return 1;
            }

            // 35us后读取数据，如果是高，说明是1；如果是低，说明是0
            delay_us(35);
            if (MASTER_READ())
            {
                data[i] |= (1 << (7 - j));
                timeout = DHT11_TIMEOUT;
                while (MASTER_READ())
                {
                    delay_us(1);
                    timeout--;
                    if (!timeout)
                        return 1;
                }
            }
        }
    }

    dht11_humid_integer = data[0];
    dht11_humid_decimal = data[1];
    dht11_temp_integer  = data[2];
    dht11_temp_decimal  = data[3];
    dht11_checksum      = data[4];

    if (dht11_checksum != (dht11_humid_integer + dht11_humid_decimal + dht11_temp_integer + dht11_temp_decimal))
    {
        return 1; // 校验和不正确
    }

    return 0; // 接收数据成功
}
```
