`Rd`：**Destination register**：目标寄存器

`Rn`：**register n**：源寄存器n

| 指令                    | 英文全称                                        | 中文解释                                     | 常用场景与说明                               |
| ----------------------- | ----------------------------------------------- | -------------------------------------------- | -------------------------------------------- |
| `mrs Rd, PSR/...`       | **Move Register from Special register**         | 从特殊寄存器读取值到普通寄存器               | 读取特殊寄存器 PSP、MSP、CONTROL、BASEPRI 等 |
| `msr PSR, Rs`           | **Move to Special Register**                    | 把通用寄存器的值写入特殊寄存器               | 写 BASEPRI 等寄存器                          |
| `mov Rd, #imm`          | **Move Immediate**                              | 把立即数赋值给寄存器                         | 初始化、赋值                                 |
| `ldr Rd, =label`        | **Load Register**                               | 加载**变量或函数地址**到寄存器               | 地址载入                                     |
| `ldr Rd, [Rn]`          | **Load Register from memory**                   | 从内存读取一个字（32位）到寄存器             | 读内存                                       |
| `str Rd, [Rn]`          | **Store Register to memory**                    | 把寄存器内容写到内存中                       | 写内存                                       |
| `stmdb Rn!, {Rlist}`    | **Store Multiple Decrement Before**             | 递减地址并压栈一组寄存器                     | 压栈保存上下文（如任务切换）                 |
| `ldmia Rn!, {Rlist}`    | **Load Multiple Increment After**               | 递增地址并从内存中恢复多个寄存器             | 出栈恢复上下文                               |
| `vstmdbeq Rn!, {Slist}` | **Vector Store Multiple Decrement Before (EQ)** | 条件执行：保存浮点寄存器（如 s16-s31）       | 保存浮点上下文                               |
| `vldmiaeq Rn!, {Slist}` | **Vector Load Multiple Increment After (EQ)**   | 条件执行：恢复浮点寄存器                     | 恢复浮点上下文                               |
| `tst Rn, #imm`          | **Test bits (AND and update flags)**            | 与立即数按位与，仅影响条件标志，不修改寄存器 | 条件检查                                     |
| `it cond`               | **If-Then**                                     | 设定条件执行的后续一条或多条指令             | Thumb 条件执行                               |
| `bl label`              | **Branch with Link**                            | 跳转到函数，并将返回地址保存到 `lr`          | 调用子程序                                   |
| `bx Rn`                 | **Branch and Exchange**                         | 跳转到寄存器 `Rn` 指向的地址（返回）         | 函数返回/切换模式                            |
| `isb`                   | **Instruction Synchronization Barrier**         | 指令同步屏障，刷新预取队列                   | 保证顺序                                     |
| `dsb`                   | **Data Synchronization Barrier**                | 数据同步屏障，确保内存写入完成               | 用于关键区域                                 |
| `nop`                   | **No Operation**                                | 什么也不做，占用一个周期                     | 调试/延迟/对齐                               |
| `cpsid i`               | **Change Processor State, Interrupt Disable**   | 禁用 IRQ 中断                                | 进入临界区                                   |
| `cpsie i`               | **Change Processor State, Interrupt Enable**    | 使能 IRQ 中断                                | 退出临界区                                   |
