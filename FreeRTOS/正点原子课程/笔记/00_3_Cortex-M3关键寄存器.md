| 寄存器名                         | 类型       | 作用简要说明                                                 |
| -------------------------------- | ---------- | ------------------------------------------------------------ |
| `PSP` (Process Stack Pointer)    | 特殊寄存器 | 每个任务使用的堆栈指针（FreeRTOS 使用 PSP 存储任务上下文）   |
| `MSP` (Main Stack Pointer)       | 特殊寄存器 | 系统异常使用的堆栈指针（如中断、硬故障等）                   |
| `LR` (Link Register) / `r14`     | 通用寄存器 | 存储返回地址或特殊值 `EXC_RETURN`，控制异常返回行为          |
| `BASEPRI`                        | 特殊寄存器 | 控制中断屏蔽阈值（高于该优先级的中断仍可进入）               |
| `CONTROL`                        | 特殊寄存器 | 控制是否使用特权级和 PSP/MSP 的切换                          |
| `PRIMASK`                        | 特殊寄存器 | 设置全局中断屏蔽标志（屏蔽所有可屏蔽中断）                   |
| `r4–r11`                         | 通用寄存器 | FreeRTOS 上下文切换中保存的任务状态寄存器（Cortex-M ABI 要求手动保存） |
| `s16–s31`                        | 浮点寄存器 | 浮点任务上下文切换时需要保存/恢复（如果使用 FPU）            |
| `xPSR` (Program Status Register) | 特殊寄存器 | 包含标志位（Zero, Carry, Negative, Overflow）、Thumb 状态等  |
