# 1.互斥量和二值信号量的区别

互斥量用于实现互斥，二值信号量用于实现同步。

根本区别在于：互斥量当其不可再take的时候，别的任务不能give。也就是说，互斥量只能由一个任务持有再释放后，另一个任务才能再持有，且任何情况不能先give再take。

# 2.中断优先级不能超过FreeRTOS可管理的最大优先级

优先级：数字越小，优先级越高

```c
void _usart_it_init(void)
{
    NVIC_InitTypeDef NVIC_InitStructure;
    NVIC_InitStructure.NVIC_IRQChannel                   = USART1_IRQn;
    NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 0;
    NVIC_InitStructure.NVIC_IRQChannelSubPriority        = 0;
    NVIC_InitStructure.NVIC_IRQChannelCmd                = ENABLE;
    NVIC_Init(&NVIC_InitStructure);

    USART_ITConfig(USART1, USART_IT_RXNE, ENABLE);
    USART_ITConfig(USART1, USART_IT_IDLE, ENABLE);
}
```

如果中断服务程序中要调用FreeRTOS相关API，则其中断优先级不能大于FreeRTOS可管理优先级。（上述程序须修改）

# 4.xSemaphoreTake(xSemaphore,xBlockTime)

其中第二个参数`xBlockTime`的含义是：如果take被阻塞，最多阻塞`xBlockTime`个tick，时间一到，就继续执行任务，take失败返回`pdFALSE`；如果在规定的时间内take成功，就返回`pdTRUE`.
