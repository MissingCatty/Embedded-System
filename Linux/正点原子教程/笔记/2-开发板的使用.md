## 1.3 网络配置

安装完成后，默认是没网的，需要自己来配置。由于是桥接模式，虚拟机成为该子网中的一个机器，独享一个ip地址。如果**连接的是校园网或者连接热点的手机连的是校园网**，这里需要手动来配置静态ip。

如果不是校园网，宿主机连接的是自家热点，直接使用DHCP即可。

【注意】：主机（即自己的windows）不能连接校园网，最好是连接手机热点。

## 1.4 ssh连接虚拟机

由于大部分项目编译需要使用命令行，而不是图形化界面，所以可以使用主机ssh连接虚拟机。

# 3.编译出厂开发镜像

## 3.1 编译出厂uboot

首先去到出厂 uboot 源码路径下  

```bash
cd ~/alpha/alientek-alpha/uboot-alientek
```

使能出厂交叉编译工具，并查看版本号

```bash
source /opt/fsl-imx-x11/4.1.15-2.1.0/environment-setup-cortexa7hf-neon-poky-linux-gnueabi
arm-poky-linux-gnueabi-gcc --version
```

【**注意**】：如果打开了新终端进行编译，也需要先使能。即使能后只有此终端下才生效，新终端无效，需要重新使能  

此时执行 build.sh 脚本即可开始编译出厂 uboot 源码。  

```
./build.sh
```
