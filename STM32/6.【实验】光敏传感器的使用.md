# 一、硬件电路

![image-20231231203601987](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20231231203601987.png)

- R2是个电位器，用于产生阈值电压
- LED1用于检测模块是否通电，LED2是检测光敏电阻是否有效（把手挡住电阻，灯灭；手移开，灯亮）
- 光照越强，光敏电阻值越小
  - 所以当**光照越强**时，N1阻值越小，AO越接近0V，所以DO输出低电平，**LED2导通发光**

## Buzzer.c

```c
#include "stm32f10x.h"                  // Device header

void Buzzer_Init(void)
{
	RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA,ENABLE);
	GPIO_InitTypeDef GPIO_InitStructure;
	GPIO_InitStructure.GPIO_Pin = GPIO_Pin_12;
	GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;
	GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
	GPIO_Init(GPIOA,&GPIO_InitStructure);
}

void Buzzer_On(void)
{
	GPIO_WriteBit(GPIOA,GPIO_Pin_12,Bit_RESET);
}

void Buzzer_Off(void)
{
	GPIO_WriteBit(GPIOA,GPIO_Pin_12,Bit_SET);
}

```

## LightSensor.c

```c
#include "stm32f10x.h"                  // Device header

void LightSensor_Init(void)
{
	RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA,ENABLE);
	GPIO_InitTypeDef GPIO_InitStructure;
	GPIO_InitStructure.GPIO_Pin = GPIO_Pin_8;
	GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IN_FLOATING;
	GPIO_Init(GPIOA,&GPIO_InitStructure);
}

uint8_t LightSensor_GetDO(void)
{
	return GPIO_ReadInputDataBit(GPIOA,GPIO_Pin_8);
}

```

## main.c

```c
#include "stm32f10x.h"                  // Device header
#include "LightSensor.h"
#include "Buzzer.h"

int main()
{	
	Buzzer_Init();
	LightSensor_Init();
	uint8_t DO;
	while(1)
	{
		DO = LightSensor_GetDO();
		if(DO)
			Buzzer_On();
		else
			Buzzer_Off();
	}
}

```
