## 示意图描述

第一张图描述外部引脚DO电压是如何产生的，是由内部的IN-和IN+两个电压通过一个电压比较器，如果IN+大于IN-，则输出1；否则输出0；

第二张图和第三张图是描述IN-，IN+和AO（模拟电压）的产生；

第四张图描述LED1和LED2的原理图。

## 工作流程

核心是第三张图里的N1光敏电阻

- 当光照变强，光敏电阻阻值变小，AO和IN+电压相同，都降低，如果降低到比IN-小时，则输出0低电平，LED2导通（说明光照强的时候，传感器上的DO指示灯亮）
- 当光站变暗，光敏电阻阻值变大，IN+升高，如果高于IN-，则始终输出1高电平，LED2断开

电位器控制光照强度输出0和1的阈值

- 将电位器阻值调大，则R2变大，IN-变小，则电压比较器阈值变小，则说明光照要比之前更强时，才能输出低电平
- 将电位器阻值调小，则R2变小，IN-变大，则电压比较器阈值变大，则说明光照比之前弱时，也能输出低电平

## 补充说明

- R2是个电位器，用于产生阈值电压，就是传感器上的那个蓝灰色旋钮
- LED1用于检测模块是否通电，LED2是检测光敏电阻是否有效（把手挡住电阻，灯灭；手移开，灯亮）

# 四、具体实现

- 光线亮，输出低电平
- 光线暗，输出高电平

则通过GPIO口即可读取传感器输出的数字量，就可以判断光照强弱。

## main.c

```c
int main(void)
{
	LED_Init();
	LightSensor_Init();

	while (1)
	{
		// 读取光敏传感器的DO
		uint8_t data = GPIO_ReadInputDataBit(LightSensor_PORT, LightSensor_PORT_PIN);

		// 如果是1，说明光线暗，打开LED
		if (data)
		{
			LED_ON();
		}
		else
		{
			LED_OFF();
		}
	}
}
```
