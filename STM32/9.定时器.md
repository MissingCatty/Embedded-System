# 1.简介

## 1.1基本功能和组成

- 基本功能：对输入时钟计数，并设定计到多少时触发计时器中断

- 组成：
  - 16位计数器：每来一个时钟，寄存器+1
  - 预分频器：对计数器的时钟进行分频
  - 自动重装寄存器：计数的目标值，即计多少个时钟申请中断

- 最大定时时间：
  - 假设：16位计数器，16位预分频器，16位自动重装寄存器，时钟频率为72MHz
  - 计$2^{16}*2^{16}$次触发一次中断，1s计72M次，所以触发一次中断最大需要$2^{16}*2^{16}/72M$秒

## 1.2定时器类型

| **类型**   | **编号**               | **总线** | **功能**                                                     |
| ---------- | ---------------------- | -------- | ------------------------------------------------------------ |
| 高级定时器 | TIM1、TIM8             | APB2     | 拥有通用定时器全部功能，并额外具有重复计数器、死区生成、互补输出、刹车输入等功能 |
| 通用定时器 | TIM2、TIM3、TIM4、TIM5 | APB1     | 拥有基本定时器全部功能，并额外具有内外时钟源选择、输入捕获、输出比较、编码器接口、主从触发模式等功能 |
| 基本定时器 | TIM6、TIM7             | APB1     | 拥有定时中断、主模式触发DAC的功能                            |

- STM32F103C8T6定时器资源：TIM1、TIM2、TIM3、TIM4

## 1.5计数器的时序

![image-20240103175655896](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240103175655896.png)

- 计数器溢出频率：CK_CNT_OV = CK_CNT / (ARR + 1)
  - ARR+1是因为计数器是从0开始数的
  - 1s计数器计数CK_CNT次，产生一次溢出计数器需要计ARR+1次，所以1s产生CK_CNT / (ARR + 1)次溢出

## 1.6计数器无预装时序和有预装时序

- 无预装：自动装入寄存器ARR无缓冲寄存器（影子寄存器）

  ![image-20240103181032162](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240103181032162.png)	当在计数过程中改变ARR的值时，**立即生效**，即：本轮计数计到新值36结束

- 有预装：ARR有缓冲寄存器

  ![](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240103181245650.png)

  真正生效的是影子寄存器，当ARR改变时，本轮计数目的值**不会立刻改变**，而是触发溢出和更新事件后的下一轮才改变。

**可以通过设置ARPE位来选择是否使用预装功能。ARPE=0时，为无预装，为1时为有预装。**

# 2.程序用法

## 2.1基本配置流程

![image-20240104170810953](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240104170810953.png)

- 打开RCC时钟

- 选择时钟源

  ```c
  /*
  	1.内部时钟源
  	2.其他定时器的时钟
  	3.捕获通道的时钟
  	4.外部时钟模式1
  	5.外部时钟模式2
  	6.配置ETR引脚的预分频、极性、滤波器
  */
  
  void TIM_InternalClockConfig(TIM_TypeDef* TIMx);
  void TIM_ITRxExternalClockConfig(TIM_TypeDef* TIMx, uint16_t TIM_InputTriggerSource);
  void TIM_TIxExternalClockConfig(TIM_TypeDef* TIMx, uint16_t TIM_TIxExternalCLKSource,
                                  uint16_t TIM_ICPolarity, uint16_t ICFilter);
  void TIM_ETRClockMode1Config(TIM_TypeDef* TIMx, uint16_t TIM_ExtTRGPrescaler, uint16_t TIM_ExtTRGPolarity,
                               uint16_t ExtTRGFilter);
  void TIM_ETRClockMode2Config(TIM_TypeDef* TIMx, uint16_t TIM_ExtTRGPrescaler, 
                               uint16_t TIM_ExtTRGPolarity, uint16_t ExtTRGFilter);
  void TIM_ETRConfig(TIM_TypeDef* TIMx, uint16_t TIM_ExtTRGPrescaler, uint16_t TIM_ExtTRGPolarity,
                     uint16_t ExtTRGFilter);
  ```

- 配置时基单元

  ```c
  void TIM_TimeBaseInit(TIM_TypeDef* TIMx, TIM_TimeBaseInitTypeDef* TIM_TimeBaseInitStruct);
  ```

- 配置输出中断控制，允许输出中断到NVIC

  ```c
  void TIM_ITConfig(TIM_TypeDef* TIMx, uint16_t TIM_IT, FunctionalState NewState);
  ```

- 配置NVIC

  - 设置优先级分组
  - 配置优先级

  ```c
  /*
  	1.设置中断优先级分组
  	2.配置优先级
  */
  
  void NVIC_PriorityGroupConfig(uint32_t NVIC_PriorityGroup);
  void NVIC_Init(NVIC_InitTypeDef* NVIC_InitStruct);
  
  typedef struct
  {
    uint8_t NVIC_IRQChannel;                    
    uint8_t NVIC_IRQChannelPreemptionPriority;
    uint8_t NVIC_IRQChannelSubPriority;
    FunctionalState NVIC_IRQChannelCmd;
  } NVIC_InitTypeDef;
  ```

- 使能定时器

  ```c
  void TIM_Cmd(TIM_TypeDef* TIMx, FunctionalState NewState);
  ```
