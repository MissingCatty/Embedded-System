# 1.简介

输出比较的作用，基本就是输出PWM波形

**何为输出比较？**

即stm32通过将计数器CNT的值与某个阈值比较，并指定当CNT比阈值大时，输出高/低电平；比阈值小时，亦然。通俗点说，就是**把定时器的一次更新的周期，划分为高低电平两个部分**。

## 1.1PWM简介

PWM（Pulse Width Modulation）脉冲宽度调制，即：高低电平宽度可以自行配置的一种脉冲。

- 在具有惯性的系统中，可以通过对一系列脉冲的宽度进行调制，来等效地获得所需要的模拟参量，常应用于电机控速等领域

- PWM参数
  - 频率 = $1 / T_S$
  - 占空比 = $T_{ON} / T_S$
  - 分辨率 = 占空比变化步距（有些波规定占空比变化1%、2%、3%...这样以1%的变化进行跳变，分辨率就是1%）

## 1.2输出比较通道

![image-20240109103942934](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240109103942934.png)

- CCRx：通道x的捕获比较寄存器（阈值）

计数器的值CNT和CCR寄存器的值比较，按照如下模式输出OC1REF

|     **模式**     |                         **描述**                         |
| :--------------: | :------------------------------------------------------: |
|       冻结       |                CNT=CCR时，REF保持为原状态                |
| 匹配时置有效电平 |                 CNT=CCR时，REF置有效电平                 |
| 匹配时置无效电平 |                 CNT=CCR时，REF置无效电平                 |
|  匹配时电平翻转  |                  CNT=CCR时，REF电平翻转                  |
|  强制为无效电平  |             CNT与CCR无效，REF强制为无效电平              |
|  强制为有效电平  |             CNT与CCR无效，REF强制为有效电平              |
|     PWM模式1     | 计数器值小于比较值时输出有效电平，大于等于时输出无效电平 |
|     PWM模式2     | 计数器值小于比较值时输出无效电平，大于等于时输出有效电平 |

- 经过CC1P选择输出是否要反向
- 最后CC1E使能输出OC1

- PWM频率：时基单元的更新频率`Freq = CK_PSC / (PSC + 1) / (ARR + 1)`
- PWM占空比：`Duty = CCR / (ARR + 1)`
- PWM分辨率：`Reso = 1 / (ARR + 1)`（占空比最小变化的步距）

# 2.程序用法

- 打开GPIO和TIM时钟

  ```c
  RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOx,ENABLE);	
  RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM2,ENABLE);		//通用定时器在APB1
  ```
  
- 配置时基单元

  ```c
  TIM_InternalClockConfig(TIMx);
  TIM_TimeBaseInitTypeDef TIM_TimeBaseInitStruct;
  TIM_TimeBaseInit(TIMx,&TIM_TimeBaseInitStruct);
  ```

- 配置输出比较单元

  ```c
  TIM_OCInitTypeDef TIM_OCInitStruct;
  TIM_OCStructInit(&TIM_OCInitStruct);	//用于给结构体赋初值，以免未赋值产生不确定的值
  TIM_OC1Init(TIMx,&TIM_OCInitStruct);
  
  typedef struct
  {
    uint16_t TIM_OCMode;			//输出比较模式
    uint16_t TIM_OutputState;		//输出比较状态，使能/失能
    uint16_t TIM_OutputNState;
    uint16_t TIM_Pulse;			//CCR的值
    uint16_t TIM_OCPolarity;		//输出比较极性
    uint16_t TIM_OCNPolarity;
    uint16_t TIM_OCIdleState;		//
    uint16_t TIM_OCNIdleState;
  } TIM_OCInitTypeDef;
  ```

  - 第2行：要么用`TIM_OCStructInit`赋初值，要么就在配置`TIM_OCInitStruct`的时候把全部结构体成员全都手动赋值，以免有不确定的成员

  **输出比较单元配置**

  | 参数              | 说明                                                         |
  | ----------------- | ------------------------------------------------------------ |
  | `TIM_OCMode`      | 输出比较模式（PWM1、PWM2、TOGGLE 等）。                      |
  | `TIM_OutputState` | 是否使能输出（`ENABLE` 或 `DISABLE`）。                      |
  | `TIM_Pulse`       | 比较值（CCR 值），决定占空比。占空比$=\frac{TIM\_Pulse}{TIM\_Period+1}$ |
  | `TIM_OCPolarity`  | 输出极性（高电平或低电平有效）。                             |

- 配置GPIO

  - TIM通道只能通过GPIO复用与外设连接

  - GPIO模式为：**复用推挽输出**，因为输出控制权已经转为了片上外设TIMx的输出通道

  ![image-20240113094701121](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240113094701121.png)

- 启动计数器

  ```c
  TIM_Cmd(TIMx,ENABLE);
  ```
