# 4.STM32F103C8T6

- 参数
  - 内核：ARM Cortex-M3
  - 主频：72MHz
  - RAM：20K（SRAM）
  - ROM：64K（Flash）
  - 供电：**2.0~3.6V**（标准3.3V）
- 命名规则
  - STM32：基于ARM核心的32位控制器
  - F：通用类型
  - 103：产品子系列
    - 101基本型
    - 102USB基本型
    - 103增强型
    - 105、107互联型
  - C：引脚数目
    - T=36
    - C=48
    - R=64
  - 8：闪存容量
    - 8=64K
  - T：封装类型
  - 6：温度范围
    - 6=-40℃~85℃
    - 7=-40℃~105℃
