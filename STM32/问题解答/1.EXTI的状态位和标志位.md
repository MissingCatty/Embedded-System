```c
FlagStatus EXTI_GetFlagStatus(uint32_t EXTI_Line);
void EXTI_ClearFlag(uint32_t EXTI_Line);
ITStatus EXTI_GetITStatus(uint32_t EXTI_Line);
void EXTI_ClearITPendingBit(uint32_t EXTI_Line);
```

上述4个函数都是对EXTI_PR（中断挂起寄存器）的读写，读取哪些位发生了中断

```c
FlagStatus EXTI_GetFlagStatus(uint32_t EXTI_Line);
ITStatus EXTI_GetITStatus(uint32_t EXTI_Line);
```

这两个函数都是读该寄存器，不同的是：

- **EXTI_GetFlagStatus**返回值是**FlagStatus**，即EXTI_Linex是否触发了中断（**EXTI_PR**）

- **EXTI_GetITStatus**返回值是**ITStatus**类型，即EXTI_Linex是否触发了中断，且未被屏蔽（**EXTI_PR + EXTI_IMR**）

所以，**EXTI_GetFlagStatus** 只是纯粹读取中断标志位的状态，但是不一定会响应中断（EXTI_IMR 寄存器对该中断进行屏蔽）；而 **EXTI_GetITStatus** 除了读取中断标志位，还查看 EXTI_IMR 寄存器是否对该中断进行屏蔽，在中断标志位置位 & 没有被屏蔽的情况下就会响应中断。
