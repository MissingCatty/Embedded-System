# 2.NVIC

- 用于管理多个中断的机构
- EXTI是用于判断哪些通道有中断触发，并上报给NVIC，NVIC对这些中断进行裁决，决定哪些中断先执行，哪些中断后执行

![image-20240102101100766](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240102101100766.png)

## 2.1响应优先级和抢占优先级

- 响应优先级：上一个**中断服务完成后**，按照响应优先级顺序决定下一个被服务的中断
- 抢占优先级：上一个**中断服务过程中**，如果此时来了另外一个中断，则**比较两个中断的抢占优先级来决定新来的中断是否能对正在进行的中断服务程序抢占CPU**（即：能否进行中断嵌套）

有时抢占优先级也被称为**主优先级**，响应优先级被称为**亚优先级**

多个任务需要被执行时，优先比较主优先级，如果主优先级相同，再比较亚优先级

## 2.2优先级分组

2.1中已经知道一个中断的优先级分为响应优先级和抢占优先级两种，NVIC提供的表示优先级的寄存器只有4位，所以**应该确定哪些位是响应优先级，哪些位是抢占优先级**

| **分组方式** | **抢占优先级**  | **响应优先级**  |
| ------------ | --------------- | --------------- |
| 分组0        | 0位，取值为0    | 4位，取值为0~15 |
| 分组1        | 1位，取值为0~1  | 3位，取值为0~7  |
| 分组2        | 2位，取值为0~3  | 2位，取值为0~3  |
| 分组3        | 3位，取值为0~7  | 1位，取值为0~1  |
| 分组4        | 4位，取值为0~15 | 0位，取值为0    |

- 数值小的优先级高

- 优先级分组是我们在写程序的时候自己配的，也就是说我们需要根据中断的数量，来确定优先级是多少位

# 3.EXTI和NVIC的关系

EXTI负责决定外部中断如何被触发，NVIC负责管理多个中断谁先上CPU执行

# 4.EXTI基本结构

![image-20240102104218275](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240102104218275.png)

- AFIO只能确定引脚的编号，所以PA0和PB0不能同时用于中断选择
- 通道数：16个GPIO、PVD、RTC、USB、ETH

- 输出数
  - 中断输出：EXTI0~EXTI4、EXTI9_5、EXTI15_10、PVD、RTC、USB、ETH（EXTI9-5共用一个中断函数，EXTI15-10共用一个中断函数）
  - 事件输出：20根信号线通向其他外设。如果外部中断被触发了，中断信号可以直接通过这20根信号线输出给其他外设，从而执行外设的某项操作（如：ADC转换、DMA）

# 5.使用方法

**开启时钟**

注意AFIO的时钟

```c
RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA, ENABLE);
RCC_APB2PeriphClockCmd(RCC_APB2Periph_AFIO, ENABLE);
```

**配置GPIO**

```c
GPIO_InitTypeDef GPIO_InitStructure;
GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IPU; // 上拉输入，默认为高电平
GPIO_InitStructure.GPIO_Pin = GPIO_Pin_10;    // PA10
GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
GPIO_Init(GPIOA, &GPIO_InitStructure);
```

**引脚映射到EXTI线上**

映射：类比于“开关”，将PA10引脚用作EXTI10的触发引脚

```c
GPIO_EXTILineConfig(GPIO_PortSourceGPIOA, GPIO_PinSource10);
```

**配置EXTI**

```c
EXTI_InitTypeDef EXTI_InitStructure;
EXTI_InitStructure.EXTI_Line = EXTI_Line10;
EXTI_InitStructure.EXTI_LineCmd = ENABLE;
EXTI_InitStructure.EXTI_Mode = EXTI_Mode_Interrupt;
EXTI_InitStructure.EXTI_Trigger = EXTI_Trigger_Falling;
EXTI_Init(&EXTI_InitStructure);
```
