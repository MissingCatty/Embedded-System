# 1.简介

## 1.1基本功能

输入捕获模式可以用来**测量脉冲宽度或者测量频率**。输入捕获模式下，当**通道输入引脚出现指定电平跳变**时，**当前CNT的值将被锁存到CCR（捕获比较寄存器）中**，可用于测量PWM波形的频率、占空比、脉冲间隔、电平持续时间等参数

**输入捕获的基本功能**：CCR寄存器捕获的是**外部输入电平的跳变**，一旦触发捕获，当前CNT的值就被锁存到CCR里。通俗点讲，就是**外部给一个触发信号，然后定时器保存当前CNT的值。**

## 1.2测量频率

- 测频法：给定一个测量时间T，在该时间内，对输入信号的上升沿/下降沿进行计次得到次数N，那么所得频率即为$$f=N/T$$

  ![image-20240114104929062](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240114104929062.png)

- 测周法：给定一个标准频率$f_{c}$，统计在**两个上升沿内**，以$f_{c}$进行计次得到次数N，则所得频率为$f=f_c/N$

  ![image-20240114104941218](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240114104941218.png)

|                  测频法                   |                       测周法                        |
| :---------------------------------------: | :-------------------------------------------------: |
|   适用于待测频率高的波形（越高越准确）    | 适用于待测频率低的波形（待测频率远低于$f_c$时最好） |
| 测量结果更新慢，且所得频率为T内**平均值** |       测量结果更新快，测量一个周期就能出结果        |
|                   平滑                    |                 受噪声影响，波动大                  |

- 高频和低频的确定：中界频率，即<u>测频法与测周法频率相等的频率点</u>，$$f_m=\sqrt{f_c/T}$$

## 1.4输入捕获通道

- 输入信号TI1首先进行滤波，得到TI1F进入边沿检测电路，得到两路输出TI1F_Rising和TI1F_Falling

- 当边沿检测检测到上升沿/下降沿时，会在对应的TI1F_Rising/TI1F_Falling通道输出一个脉冲，CC1P寄存器选择是看上升沿脉冲还是下降沿脉冲

  ![常用数字电路模块：边沿检测电路](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/v2-b3b277cb537454af04640d18f114aab9_1440w.image)

- TI1FP1一路通到从模式控制器，另一路通到分频器

- 分频器即来N个脉冲才进行一次捕获

## 1.5定时器主从模式

- 主模式：定时内内部信号映射到TRGO引脚，用于触发别的外设（**主动输出信号，控制其他外设**）
- 从模式：接收其他外设或自身信号，用于控制定时器自身的运行（**被动接收信号，控制自身运行**）

## 1.6输入捕获测频率和占空比

- 测频率（一个捕获比较器，只捕获上升沿）

  ![image-20240115181616168](https://zyc-learning-1309954661.cos.ap-nanjing.myqcloud.com/machine-learning-pic/image-20240115181616168.png)

  CNT初值为0，在一个上升沿来临的时候，捕获CNT的值入CCR，之后将CNT值清零，此时CCR中就是CNT的计次N

  - 这种方法测量的是两个上升沿之间的计数次数，所以在**上升沿被捕获的同时，要将计数器CNT的值清零**。清零可以是手动清零，也可以是**硬件自动清零（通过配置从模式可以实现，1.4图中的TI1FP1可实现从模式的配置）**

- 测占空比（两个捕获比较器，一个捕获上升沿，一个捕获下降沿）
