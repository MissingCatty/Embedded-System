# 要求：用控制寄存器的方式，点亮PC13口的灯

```c
int main(void)
{
	RCC->APB2ENR = 0x00000010;	//打开GPIOC的时钟
	GPIOC->CRH = 0x00300000;
	GPIOC->ODR = 0x00000000;	//GPIO数据寄存器，给GPIOC的13号口输出一个高电平（低电平点亮）
	while(1)
	{
		
	}
}
```

- PC13：GPIOC的13号口

- Led**低电平点亮**

# 用库函数方式点灯

```c
int main(void)
{
	RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOC,ENABLE);	//GPIOC时钟使能
    
	GPIO_InitTypeDef GPIO_InitStructure;					//定义结构体，配置GPIO
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_PP;	//推挽输出
	GPIO_InitStructure.GPIO_Pin	= GPIO_Pin_13;			//13号引脚
	GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;	//输出速率
	GPIO_Init(GPIOC,&GPIO_InitStructure);
	GPIO_SetBits(GPIOC, GPIO_Pin_13);						//PC13置高电平
	GPIO_ResetBits(GPIOC, GPIO_Pin_13);						//PC13置低电平
	
	while(1)
	{
		
	}
}
```
